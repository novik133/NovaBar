# Bluetooth Indicator Keyboard Shortcuts

## Global Shortcuts

- **Ctrl+B**: Toggle Bluetooth power (enable/disable adapter)
- **Ctrl+S**: Start/stop device scan
- **Ctrl+L**: Connect to last connected device
- **Escape**: Close Bluetooth popover

## Navigation Shortcuts

### General Navigation
- **Tab**: Move focus to next element
- **Shift+Tab**: Move focus to previous element
- **Enter/Space**: Activate focused button or list item

### Device List Navigation
- **Up Arrow**: Select previous device in list
- **Down Arrow**: Select next device in list
- **Home**: Select first device in list
- **End**: Select last device in list
- **Page Up**: Move up 5 devices in list
- **Page Down**: Move down 5 devices in list
- **Menu/F10**: Show context menu for selected device

## Accessibility Features

### Screen Reader Support
- All interactive elements have ARIA labels and roles
- State changes are announced (device connected, pairing started, etc.)
- Error messages are announced with appropriate priority

### Keyboard Navigation
- Complete keyboard navigation through all UI elements
- Visible focus indicators on all interactive elements
- Tab order follows logical flow

### Visual Indicators
- Focus indicators show which element has keyboard focus
- High contrast support for better visibility
- Text alternatives for all icons and visual indicators

## Usage Tips

1. **Quick Bluetooth Toggle**: Press Ctrl+B to quickly enable or disable Bluetooth without opening the popover
2. **Fast Scanning**: Press Ctrl+S to start scanning for devices
3. **Reconnect Quickly**: Press Ctrl+L to reconnect to your most recently used device
4. **Navigate Efficiently**: Use arrow keys to quickly browse through available devices
5. **Context Actions**: Press Menu or F10 on a selected device to access trust, block, and forget options
